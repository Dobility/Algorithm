## 题目描述

修改 js 代码中 parseInt 的调用方式，使之通过全部测试用例

```js
function parse2Int(num) {
    return parseInt(num);
}
```



示例1

## 输入

[复制](javascript:void(0);)

```
'12'
```

## 输出

[复制](javascript:void(0);)

```
12
```
